\chapter{Muskrat Trap}
\hypertarget{index}{}\label{index}\index{Muskrat Trap@{Muskrat Trap}}
This project is a simple example of a payload encoder and decoder for a Lo\+Ra\+WAN application. The project consists of two classes\+: \doxylink{classpayload_encoder}{payload\+Encoder} and \doxylink{classpayload_decoder}{payload\+Decoder}. The \doxylink{classpayload_encoder}{payload\+Encoder} class encodes variables into a single payload, and the \doxylink{classpayload_decoder}{payload\+Decoder} class decodes the payload back into the original variables. The project also contains a unit test that tests the encoder and decoder classes.
