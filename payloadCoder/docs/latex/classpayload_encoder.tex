\doxysubsection{Detailed Description}
payload endoder class 

This class wil encode variables for the Lo\+Ra\+WAN application in to a single payload The class is setup using both .h and .cpp files where the setters and getters are placed in to the .cpp file. The {\ttfamily \doxylink{classpayload_encoder}{payload\+Encoder}} class is responsible for encoding payload data that includes various sensor readings and status information. It provides methods to set individual data fields and compose the payload. The encoded payload can be retrieved using the {\ttfamily \doxylink{classpayload_encoder_a5eb6a363dc1e5c8bdae8e8d2848246fa}{get\+Payload()}} method. The size of the payload can be obtained using the {\ttfamily \doxylink{classpayload_encoder_a279316cc9471f69d677ae96663f5db4a}{get\+Payload\+Size()}} method. The encoded payload can be printed in binary format using the {\ttfamily \doxylink{classpayload_encoder_a7f571901d7ab6265341e2025fb521c6d}{print\+Payload\+Binary()}} method. The encoded payload can be printed in a human-\/readable format using the {\ttfamily \doxylink{classpayload_encoder_a2d177e35b0f8e1cf510a41a29e3016b8}{print\+Payload\+Encoded()}} method.
